TIMER:
SET TIMER BY PRESSING A
START TIMER BY PRESSING B
IF TIMER IS RUNNING, IT WILL SHOW A SAD FACE
IF TIMER IS UP, IT WILL SHOW A SMILEY FACE


MICRO-BIT TIMER:
- SHOWS A LOADING PHASE WHEN TIMER IS SET TO MORE THAN 5 SECONDS
- SHOWS THE TIME ELAPSED WHEN TIME IS LESS THAN 5 SECONDS
- GIVES THE NYAN CAT MEME SOUND WHEN TIMER IS UP
